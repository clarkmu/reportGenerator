This project is also available on GitHub

To Use:
  1. Select charts to place in PDF
  2. Select .csv file (Charts expect first column to be words, all others to be number values)
  3. Drag and drop charts
  4. Right click page, select "Print...", select "Save as PDF" (This box does not appear in PDF)
